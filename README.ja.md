# RealisticChicken

RealisticChicken は、軽量な Paper/Spigot プラグインで、ニワトリの繁殖をよりリアルにします。
有精卵を使ってヒヨコを孵化させ、無精卵は自動でドロップされます。

## 特徴

- 有精卵はヒヨコを孵化させることができます。
- 無精卵は自動でドロップされます。
- 繁殖時にもバニラ相当の経験値（1〜7XP）が得られます。
- ニワトリは繁殖後、5分のクールダウンに入ります。
- Paper、Spigot、Bukkit に対応。
- Minecraft 1.21.x に対応。
- 依存プラグイン不要。
- 卵の説明は複数言語に対応。

## 仕組み

**バニラ Minecraft の場合:**
- 種を与える → ヒヨコが即座に出現。
- 卵は自動で産まれる。

**RealisticChicken:**
- 種を与える → 卵を産む。
- 有精卵を投げるとヒヨコが孵化。
- 無精卵は時間経過で自動ドロップ。

よりリアルで没入感のある農業体験を提供します。

## インストール方法

1. プラグインの JAR をダウンロード。
2. サーバーの `/plugins/` フォルダに配置。
3. サーバーを再起動。
